  1. Log in to Hostaway as an admin.
  2. Go to Settings → Hostaway API.
  3. Create a new Hostaway API Key.
  4. Enter a Name [HostScout] Partner [Hostaway Public API] Press [Create]
  5. Copy the Account ID and API Key into this form.
